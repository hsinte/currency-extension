# 🇹🇼 臺灣銀行即時匯率換算器 (Chrome Extension)

一個輕量、流暢且完全動態的 Chrome 瀏覽器擴充功能。直接串接臺灣銀行最新牌告匯率開放資料，提供最精確的**現金賣出價**即時雙向換算，並隨附近 30 天的歷史匯率趨勢圖，是您出國旅遊、外幣投資的隨身好幫手。

---

## ✨ 核心特色

- 📊 **動態幣別載入**：自動解析台銀當日最新 CSV，幣別清單同步更新，免人工維護。
- 💱 **現金賣出雙向聯動**：一律以現金賣出價為計算基準。不論切換上方看板、或是調整換算選單，上下兩區塊皆會**全自動智慧同步連動**。
- 📈 **近 30 天趨勢折線圖**：一鍵展開美觀的動態折線圖（Chart.js），精準掌握外幣近一個月的走勢脈絡。
- ⚙️ **專屬預設值設定**：提供獨立的 Options 設定網頁，可自訂每次點開擴充功能時預設想查看的主力外幣。
- 🎨 **優雅精緻 UI**：全面採用 TailwindCSS 打造現代、乾淨且直覺的操作介面。

---

## 🛠️ 開發與建置

本專案使用 **TypeScript** 與 **Vite** 進行模組化建置。

### 1. 安裝環境依賴

```bash
npm install
```

### 2. 開發預覽編譯（隨動更新）

```bash
npm run dev
```

### 3. 生產環境打包

```bash
npm run build
```

打包完成後，專案根目錄會生成一個 `dist/` 資料夾，這就是完整的擴充功能封裝包。

---

## 📦 如何安裝到 Chrome 瀏覽器

1. 打開 Chrome 瀏覽器，在網址列輸入 `chrome://extensions/` 進入擴充功能管理頁面。
2. 開啟右上角的 **「開發者模式 (Developer mode)」** 切換開關。
3. 點擊左上角的 **「載入解壓縮擴充功能 (Load unpacked)」** 按鈕。
4. 選擇本專案打包好的 **`dist`** 資料夾，即可成功載入！
5. 建議在瀏覽器工具列點擊「拼圖」圖示，將「台銀即時匯率換算器」**釘選 (Pin)** 到畫面上方便隨時點閱。
